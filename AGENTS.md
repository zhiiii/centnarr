# AGENTS.md

This file is read by AI agents (Claude Code, Cursor, etc.) at the start of every session to understand how to work in this repo.

## Agent skills

### Issue tracker

Issues live as GitHub issues at `github.com/zhiiii/centnarr`. See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage roles map 1:1 to their default label strings. See `docs/agents/triage-labels.md`.

### Domain docs

This is a **multi-context** repo (frontend + backend). A `CONTEXT-MAP.md` and per-context `CONTEXT.md` files will live at the repo root and inside each context directory. See `docs/agents/domain.md`.
